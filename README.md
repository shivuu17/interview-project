# Mall of America — Interactive Sales Deck

This is a production-ready scaffold for a cinematic, premium sales deck web app built with React + Vite, Tailwind CSS, Framer Motion, GSAP, Swiper, and Three.js.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run locally

```bash
npm run dev
```

Build

```bash
npm run build
```

Deployment

Deploy to Vercel by connecting the repository and using the `npm run build` output. Use environment `NODE_ENV=production`.

Notes

- Placeholder assets use free sample videos and Unsplash images. Replace with studio-grade media for production.
- For mapping/audience reach, integrate Mapbox or custom SVG visualizations.
- Aim for Lighthouse 90+ by optimizing images, using modern formats, and adding server-side compression and CDN.
